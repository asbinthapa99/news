# PyCast — Automated Python News

A sleek, modern web application for discovering and staying updated with the latest Python programming news and updates.

## Features

- **Real-time News Feed**: Curated Python news and updates from across the ecosystem
- **Clean, Modern UI**: Minimalist dark-themed interface with smooth animations
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Professional Typography**: Uses curated font families for optimal readability
- **Interactive Elements**: Smooth transitions and hover effects for engaging user experience

## Tech Stack

- **Frontend**: HTML5, CSS3
- **Design**: Custom CSS with grid noise texture and animated gradients
- **Fonts**: Curated selection of Google Fonts (Fraunces, Syne, DM Mono)
- **Styling**: Dark theme with accent colors (Lime Green: #c8f135, Mint: #3bffbd)

## Getting Started

Simply open `index.html` in your web browser to view the PyCast news feed.

## Color Scheme

- **Background**: Deep black (#0d0e0f)
- **Surface**: Dark gray (#141516)
- **Primary Accent**: Lime green (#c8f135)
- **Secondary Accent**: Mint green (#3bffbd)
- **Text**: Light gray (#e8eaed)
