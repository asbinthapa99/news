# Daily news pipeline — runs at 06:00 UTC
from pycast import scrape, summarise, broadcast
 
def run_pipeline():
    sources = scrape("peps,pypi,discourse,hn")
    digest = summarise(sources, model="claude")
    episode = broadcast(digest, fmt="video+rss")
    return episode

# What's under the hood

01 ——
🕷️
Multi-Source Scraper

Crawls PyPI releases, PEPs, Hacker News, Python Discourse, BBC, Reuters, and major dev blogs daily using feedparser + httpx.

02 ——
🧠
LLM Curation

An AI layer ranks, de-dupes and summarises each story into a tight two-sentence digest. Hallucination-checked against raw sources.

03 ——
🎙️
Voice & Video Synthesis

ElevenLabs TTS reads the script. FFmpeg compiles b-roll, code snippets, and captions into a finished 5-minute YouTube video.

04 ——
📡
Auto-Broadcast

YouTube Data API, RSS feed, and a newsletter are all published in a single atomic step before 07:00 UTC every morning.

05 ——
🔍
Full Transparency

Every episode links to its raw sources and prompt logs. We believe in showing our work — reproducibly and openly.

06 ——
⚙️
Contribute & Fork

The entire pipeline is MIT-licensed on GitHub. Add a new source, tweak the ranking model, or deploy your own edition in under 20 minutes.
